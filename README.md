# bookstore-app
“A full-stack Bookstore Management System built with React (Vite), Node.js, Express, and MySQL. Features include book listings, CRUD operations, and a responsive UI.”

## 🚀 Features

- 📖 List all books
- ➕ Add new books
- ✏️ Edit existing book details
- ❌ Delete books

**Frontend:**
- React + Vite
- Axios 

**Backend:**
- Node.js
- Express.js
- MySQL
